# My Edge Extension
自制 Edge 扩展
## 功能
- **csdn**：阻止登录界面，允许不登录复制，自动展开隐藏的代码，自动展开关注后查看全文。
- **淘宝教育**：移除跑马灯版权声明。
- **简书**: 移除下载客户端的二维码弹窗
- **哔哩哔哩**: 增加更多播放视频时的倍速选择
- **Chunk Base**: ~~作为一名 mc 玩家~~右键地图时自动复制当前点 `/tp` 命令

## 环境
**Microsoft Edge**

## 安装
1. 管理扩展-开启**开发人员模式**
2. 加载解压缩的扩展，选择扩展文件夹（manifest.json 所在目录）
